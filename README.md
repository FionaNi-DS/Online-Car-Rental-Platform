# Online-Car-Rental-Platform
A Python-based online car rental system that allows users to rent and return cars on an hourly, daily, or weekly basis, with automatic billing and stock management.

Implements **object-oriented design**, real-time **stock management**, and **automated billing**.

## Features
- Rent cars hourly, daily, or weekly
- Return cars with automatic bill calculation
- Stock validation and updates
- Modular structure:  
  - `car_rental.py` → Core logic & classes  
  - `main.ipynb` → Interactive menu interface

## Tech Highlights
- Python OOP (classes, encapsulation)
- `datetime` for time-based billing
- Clean modular design for scalability

## Files
- `car_rental.py`
- `main.ipynb`
